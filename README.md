# Portfolio Content Starter

This folder setup provides the essential markdown files for a complete portfolio.

## Files

- `content/background.md` - Your intro, story, values, and goals.
- `content/practices.md` - Your workflow, principles, and engineering/design practices.
- `content/experience.md` - Work history and responsibilities.
- `content/projects.md` - Selected projects with outcomes and tech stack.
- `content/skills.md` - Skills grouped by area and proficiency.
- `content/contact.md` - Contact details and availability.

## Suggested Usage in Astro

1. Map these markdown files into your portfolio pages.
2. Keep writing concise and outcome-focused.
3. Update projects and practices every month.

## Quick Start

1. Fill each file section with your own details.
2. Replace placeholders in brackets.
3. Use short bullet points for readability.
